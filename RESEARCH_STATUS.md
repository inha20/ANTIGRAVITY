## Repository Standards

Target structure for each paper repository:

```
Repository/
├── README.md           # Standard format with Abstract, Keywords, Research Context
├── HANDOVER.md         # Continuation guide
├── CHANGELOG.md        # Version history
├── paper/              # Main thesis/paper files
├── figures/            # Standardized figures (FigureNN_description.png)
│   └── originals/      # Original files preserved here
├── references/         # References and citations
└── notes/              # Working notes
```
